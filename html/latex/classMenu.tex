Function to switch a student from one class/\+UC to another. 

Switch a student from one class or UC to another.

This function allows the user to switch a student from their current class or UC to another class or UC. It prompts the user to choose between switching within UCs or within classes, takes the student\textquotesingle{}s code, current UC or class code, and the target UC or class code for the switch. It validates the availability and occupancy of the target class, checks for class overlaps, and updates the student\textquotesingle{}s schedule accordingly. It also updates the class information in the data file and logs the switch action in the history.

\begin{DoxyNote}{Note}
The function provides error handling for various scenarios, including class availability, class occupancy, and class overlaps, and allows the user to return to the previous menu or exit the program.
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
